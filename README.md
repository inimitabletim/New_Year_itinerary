# 2026 北投新春之旅 PWA

這是一個專為 2026 農曆春節北投家庭旅遊設計的**響應式網頁應用程式 (PWA)**。

## 重點功能

- **倒數計時**：隨時掌握距離出發 (2/17) 還有多久。
- **每日行程**：分頁切換初一至初四行程，清晰易讀。
- **智能導航**：點擊地點即可開啟 Google Maps 導航。
- **手機安裝**：支援「加入主畫面」，像 APP 一樣使用 (無需下載)。

## 如何觀看

### 方法一：直接在電腦開啓 (快速預覽)

1.  雙擊 `index.html` 即可在瀏覽器中開啟。
2.  (推薦) 若有安裝 VS Code，可使用 "Live Server" 套件開啟，體驗更佳。

### 方法二：在手機上觀看 (GitHub Pages)

1.  將本專案推送到 GitHub (已完成)。
2.  進入 GitHub Repository -> **Settings** -> **Pages**。
3.  在 "Build and deployment" 下的 Source 選擇 **Deploy from a branch**。
4.  Branch 選擇 **main**，folder 選擇 **/(root)**，按 Save。
5.  等待約 1-2 分鐘，GitHub 會給您一個網址 (例如 `https://您的帳號.github.io/New_Year_itinerary/`)。
6.  用手機打開該網址，並**加入主畫面**，即可離線瀏覽部分內容。

## 開發指令

- 安裝依賴：`npm install`
- 本地開發：`npm run dev`
